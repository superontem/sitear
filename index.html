    Linguagens de Baixo Nível:

    Essas linguagens estão mais próximas do hardware, ou seja, oferecem
    pouca ou nenhuma abstração.
    Elas permitem que os programadores tenham um controle mais direto sobre
    os componentes do computador, como memória e processador.

  • C+++

    • É uma linguagem de alto nível orientada a objetos que é uma extensão da
    linguagem C. C++ oferece tanto abstração de alto nível (como classes e
    objetos) quanto a possibilidade de trabalhar em um nível mais próximo do
    hardware, se necessário. Muito utilizada no desenvolvimento de sistemas
    operacionais, jogos, motores gráficos e softwares de alto desempenho.

  • Java

    • É uma linguagem de alto nível, orientada a objetos, muito popular para
    desenvolvimento de aplicativos web, móveis e empresariais. O código Java
    é executado em uma máquina virtual (JVM), o que permite que seja
    altamente portátil entre diferentes sistemas operacionais.

  • Vantagens de Linguagens de Alto Nível:

    • Facilidade de uso, com sintaxe mais próxima da linguagem humana.

    Mais produtivas para os desenvolvedores, pois requerem menos código para
    realizar tarefas complexas.

    Maior portabilidade: podem ser usadas em diferentes plataformas com
    pouca ou nenhuma modificação.


  • Desvantagens:

    • Menor controle sobre o hardware em comparação com linguagens de baixo nível.

    Algumas linguagens de alto nível podem ser mais lentas ou ineficientes, embora linguagens como C++ tentem minimizar esse impacto.
